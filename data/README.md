# Data

Data files are not tracked in GitHub.

Download the project data from Zenodo:

[https://doi.org/10.5281/zenodo.19674252](https://doi.org/10.5281/zenodo.19674252)

Place the required input file here:

```text
data/all_locations.csv
```

The workflow expects `data/all_locations.csv` to include these columns:

```text
individual_local_identifier
X
Y
timestamp
study_site
study_name
ground_speed
deploy_on_timestamp
```

Derived `.rds` files are generated by `Rscript run_workflow.R` and are also ignored by Git.
